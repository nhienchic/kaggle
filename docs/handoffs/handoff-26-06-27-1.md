## Product Decisions Already Made

Build a Streamlit app called Kaggle Capstone Submission Coach.

The app should help finish the Kaggle AI Agents capstone submission by reading the competition brief and repo, then producing a submission readiness report and supporting artifacts.

Accepted architecture:

- Streamlit web app
- Multi-agent pipeline
- Four specialist agents: Requirement Analyst, Repo Auditor, Submission Strategist, Communication Coach
- Gemini / Google ADK intended runtime, gated by `GOOGLE_API_KEY`
- Deterministic fallback when no key is configured
- MCP-compatible tool layer
- Security scan for likely secrets and safe read boundaries
- Deployability documentation

Accepted MVP boundary:

- Analyze and generate artifacts only
- Do not automatically edit the target repo
- Do not automatically create GitHub issues from the generated build plan in v1
- Do not require live LLM calls for the demo to work
