## Important Context

Two deployment-readiness issues were fixed during this wrap-up:

1. Streamlit Cloud secrets/runtime mounts could resolve outside the repository root and crash the security scan. Fixed in `kaggle_capstone_coach/security.py` by skipping outside-resolving runtime files during repository scanning while preserving direct `safe_read_text()` rejection behavior.
2. The regression test initially contained a fake literal shaped like a committed API key. The app correctly flagged it as a likely committed secret. Fixed in `tests/test_security_scanner.py` by constructing the fixture dynamically so the source no longer contains a secret-like assignment.

The README has been sharpened into the main judge-facing repository document. It now includes live links, concept map, architecture, quick start, MCP commands, Gemini configuration, security model, deployment notes, submission assets, limitations, and final status.

Do not commit real API keys. Use local environment variables or Streamlit secrets for `GEMINI_API_KEY` / `GOOGLE_API_KEY`.
